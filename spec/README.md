# フォルダ構成

## ubi
ゲーム内で使用する単語を説明する辞書です（ユビキタス言語）

## ux
ゲームが目指す目標、ゴール、コンセプトをここに示します

## usecase
ゲーム内のふるまいを定義します（Gherkin形式）

## rule
AIの行動コード生成やゲーム全体にかかわるルールが記載されます
- general.md: 全般ルール
- classes.md: クラス設計ルール
- Input.md: 入力定義
- coding.md: コーディング規約
- gamerule.md: ゲームルール

## gamedesign
ゲーム内の仕様、レベルデザインにまつわる仕様書が入ります
- bottle.md: ボトル定義
- gacha.md: ガチャ確率
- coin.md: コイン経済

## unityWeb
Webアプリ（スマホ側）の画面・機能仕様

## unityMain
母艦アプリの画面・機能仕様

## shared
両アプリで共有するデータモデル・通信プロトコル

## test
テスト仕様
- feature.md: 機能テスト
- performance.md: パフォーマンステスト

## image
説明に必要な画像をここに入れていきます
