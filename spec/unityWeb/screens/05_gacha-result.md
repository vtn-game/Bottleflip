# ガチャ演出・結果画面

## 画面ID
`WEB-SCR-005`

## 概要
ガチャを引いた後の演出と結果を表示する画面。

## 表示条件
- ガチャ画面で「引く」ボタン押下後

## UI要素

### 演出フェーズ

| 要素ID | 種類 | 説明 |
|--------|------|------|
| container_effect | Container | 演出エフェクトコンテナ |
| anim_pachinko | Animation | パチンコ風演出アニメーション |
| particles | ParticleSystem | パーティクルエフェクト |

### 結果表示フェーズ

| 要素ID | 種類 | 説明 |
|--------|------|------|
| img_bottle | Image | 獲得ボトル画像 |
| txt_bottle_name | Text | ボトル名 |
| txt_rarity | Text | レア度 |
| txt_new | Text | 「NEW!」（新規獲得時） |
| txt_dupe | Text | 「+50コイン」（被り時） |
| btn_ok | Button | 「OK」ボタン |

## 動作仕様

### 演出フェーズ
1. レア度に応じた演出を再生
   - ★1〜2：通常演出（1秒）
   - ★3：ちょっと豪華演出（2秒）
   - ★4：かなり豪華演出（3秒）
   - ★5：超豪華演出（4秒）
2. 演出中はタップでスキップ可能

### 結果表示フェーズ
1. ボトル情報を表示
2. 新規獲得の場合：「NEW!」表示
3. 被りの場合：コインに変換して表示

### OKボタン
- ガチャ画面へ遷移

## 遷移先
- → ガチャ画面

## 演出パターン

### 通常演出（★1〜2）
- シンプルなガチャ玉が転がる
- 軽いエフェクト

### 豪華演出（★3）
- 金色のエフェクト追加
- ドラム音

### かなり豪華演出（★4）
- 虹色エフェクト
- 確変風演出

### 超豪華演出（★5）
- フルスクリーンエフェクト
- 大当たり演出
- 専用BGM
