# ガチャ画面

## 画面ID
`WEB-SCR-004`

## 概要
3台のガチャ機からボトルを入手する画面。

## 表示条件
- メイン画面から「ガチャ」ボタン押下時
- 初回プレイ時（名前入力後）

## UI要素

| 要素ID | 種類 | 説明 |
|--------|------|------|
| txt_title | Text | 「ガチャ」 |
| txt_coin | Text | 所持コイン数 |
| gacha_machine_1 | Prefab | ガチャ機1 |
| gacha_machine_2 | Prefab | ガチャ機2 |
| gacha_machine_3 | Prefab | ガチャ機3 |
| btn_back | Button | 「もどる」ボタン |

### ガチャ機（gacha_machine）構成

| 要素ID | 種類 | 説明 |
|--------|------|------|
| img_machine | Image | ガチャ機画像 |
| txt_machine_name | Text | ガチャ機名 |
| txt_remaining | Text | 残り回数「残り XX/YY」|
| btn_contents | Button | 「中身を見る」ボタン |
| btn_pull | Button | 「100コインで引く」ボタン |
| scroll_preview | ScrollView | 中身プレビュー（展開時） |

## 動作仕様

### ガチャ機表示
- 3台のガチャ機を横並びで表示
- 各ガチャ機の残り回数をリアルタイム表示
- 残り0の場合は「SOLD OUT」表示

### 中身を見るボタン
- 該当ガチャ機の中身一覧をプレビュー表示
- 既に排出されたものはグレーアウト

### 引くボタン
1. 所持コイン確認（100コイン必要）
2. 不足時：「コインが足りないよ！」表示
3. 十分な場合：
   - サーバーへガチャリクエスト送信
   - 演出再生開始
   - 結果表示画面へ

### もどるボタン
- メイン画面へ遷移

## 遷移先
- → ガチャ演出画面
- → メイン画面

## 備考
- ボックスガチャ（一番くじ方式）のため、引いたアイテムは減少する
- 全て引き切ると補充される（リセット）
