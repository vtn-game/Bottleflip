# 名前入力画面

## 画面ID
`WEB-SCR-001`

## 概要
初回接続時にプレイヤー名を入力する画面。

## 表示条件
- 初回起動時
- ローカルストレージにプレイヤー情報がない場合

## UI要素

| 要素ID | 種類 | 説明 |
|--------|------|------|
| txt_title | Text | 「名前を入力してね！」 |
| input_name | InputField | プレイヤー名入力欄（最大12文字） |
| btn_start | Button | 「はじめる」ボタン |

## 動作仕様

### 名前入力
- 最大12文字まで入力可能
- 空文字は不可
- 絵文字は使用可能

### はじめるボタン押下時
1. 入力値のバリデーション
2. サーバーへプレイヤー登録リクエスト送信
3. 登録成功時：100コイン付与 → ガチャ画面へ遷移
4. 登録失敗時：エラーメッセージ表示

## 遷移先
- → ガチャ画面（初回のみ）

## 備考
- 2回目以降の起動時はこの画面をスキップしメイン画面へ直接遷移
