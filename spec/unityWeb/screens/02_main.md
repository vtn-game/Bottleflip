# メイン画面（ボトル投げ画面）

## 画面ID
`WEB-SCR-002`

## 概要
スマホを振ってボトルを投げるメイン画面。
2回目以降の起動時はここに直接遷移する。

## 表示条件
- 名前入力完了後
- 2回目以降の起動時

## UI要素

| 要素ID | 種類 | 説明 |
|--------|------|------|
| txt_player_name | Text | プレイヤー名表示 |
| txt_coin | Text | 所持コイン数 |
| img_current_bottle | Image | 現在選択中のボトル画像 |
| txt_bottle_name | Text | ボトル名 |
| txt_hint | Text | 「スマホを振って投げよう！」 |
| btn_change_bottle | Button | 「ボトルを変更」ボタン |
| btn_gacha | Button | 「ガチャ」ボタン |
| indicator_shake | Animation | 振り検知インジケーター |

## 動作仕様

### スマホ振り検知
1. 加速度センサーで振りを検知
2. しきい値を超えた場合、振りの強さを計算
3. 母艦アプリに加速度データを送信
4. コメント入力画面へ遷移

### ボトルを変更ボタン
- ボトル選択画面へ遷移

### ガチャボタン
- ガチャ画面へ遷移

## 遷移先
- → ボトル選択画面
- → ガチャ画面
- → コメント入力画面（投げた後）

## 演出
- 振りを検知するとインジケーターがリアクション
- ボトルが少し揺れるアニメーション
