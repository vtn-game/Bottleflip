# ボトル選択画面

## 画面ID
`WEB-SCR-003`

## 概要
所持しているボトル一覧から使用するボトルを選択する画面。

## 表示条件
- メイン画面から「ボトルを変更」ボタン押下時

## UI要素

| 要素ID | 種類 | 説明 |
|--------|------|------|
| txt_title | Text | 「ボトルを選ぼう」 |
| scroll_bottles | ScrollView | ボトル一覧（縦スクロール） |
| item_bottle | Prefab | ボトルアイテム（下記参照） |
| btn_back | Button | 「もどる」ボタン |

### ボトルアイテム（item_bottle）構成

| 要素ID | 種類 | 説明 |
|--------|------|------|
| img_bottle | Image | ボトル画像 |
| txt_name | Text | ボトル名 |
| txt_rarity | Text | レア度（★表示） |
| txt_difficulty | Text | 難易度（かんたん/ふつう/むずかしい/げきむず） |
| txt_coin | Text | 獲得コイン量 |
| img_selected | Image | 選択中マーク |

## 動作仕様

### ボトル一覧表示
- 所持しているボトルのみ表示
- レア度順にソート（高い順）
- 現在選択中のボトルには選択マーク表示

### ボトル選択
1. ボトルアイテムをタップ
2. 選択中ボトルを変更
3. サーバーへ選択情報を保存
4. 選択マーク更新

### もどるボタン
- メイン画面へ遷移

## 遷移先
- → メイン画面

## ボトル情報表示フォーマット

```
┌─────────────────────────┐
│  [ボトル画像]           │
│  ペットボトル           │
│  ★★☆☆☆               │

│  コイン：+50            │
└─────────────────────────┘
```
