# コメント入力画面

## 画面ID
`WEB-SCR-006`

## 概要
ボトル投げ後にコメントを残す画面。

## 表示条件
- ボトル投げ完了後

## UI要素

| 要素ID | 種類 | 説明 |
|--------|------|------|
| txt_result | Text | 「成功！」or「ざんねん...」 |
| txt_coin_earned | Text | 獲得コイン（成功時のみ） |
| txt_prompt | Text | 「コメントを残そう！」 |
| input_comment | InputField | コメント入力欄（1行、最大30文字） |
| btn_post | Button | 「投稿する」ボタン |
| btn_skip | Button | 「スキップ」ボタン |

## 動作仕様

### 結果表示
- 成功時：「成功！ +○○コイン」
- 失敗時：「ざんねん...」

### コメント入力
- 最大30文字
- 絵文字使用可能
- 空文字でも投稿可能（その場合ボトルは消える）

### 投稿するボタン
1. コメント内容をサーバーへ送信
2. 母艦アプリでボトル上にコメント表示
3. メイン画面へ遷移

### スキップボタン
1. コメントなしでサーバーへ通知
2. 母艦アプリでボトルが消える
3. メイン画面へ遷移

## 遷移先
- → メイン画面
