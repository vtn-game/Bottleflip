# コイン経済

## 概要
ゲーム内通貨「コイン」の獲得と消費のバランス設計

## 獲得手段

### ボトルフリップ成功
| 条件 | コイン |
|------|--------|
| 通常成功 | +50 |
| ボーナスボトル使用時 | +50 × 倍率 |

### その他（TODO: 将来実装）
| 手段 | コイン |
|------|--------|
| デイリーログイン | +10〜50 |
| 連続成功ボーナス | +10（2連続）〜+100（10連続） |
| イベント報酬 | 変動 |

## 消費手段

| アクション | コイン |
|-----------|--------|
| 単発ガチャ | -100 |
| 10連ガチャ | -900 |

## 経済バランス

### 想定プレイサイクル
```
投げ成功率: 約30%想定

10回投げる → 3回成功 → +150コイン
→ 約7回のサイクルで単発ガチャ1回分

100回投げる → 30回成功 → +1500コイン
→ 10連ガチャ1回分 + 単発6回分
```

### コイン獲得効率
| プレイ回数 | 成功数（30%） | 獲得コイン | ガチャ可能回数 |
|-----------|--------------|-----------|---------------|
| 10回 | 3回 | 150 | 単発1回 |
| 50回 | 15回 | 750 | 単発7回 |
| 100回 | 30回 | 1500 | 10連1回+単発6回 |
