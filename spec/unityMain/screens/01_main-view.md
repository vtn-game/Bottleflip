# メインビュー画面

## 画面ID
`MAIN-SCR-001`

## 概要
ボトルフリップが行われるメイン画面。投げられたボトルとコメントが表示される。

## 表示条件
- アプリ起動時（常時表示）

## UI要素

### 3Dビュー領域

| 要素ID | 種類 | 説明 |
|--------|------|------|
| scene_main | 3D Scene | メイン3Dシーン |
| floor | 3D Object | 床（着地判定用） |
| table | 3D Object | テーブル（ボトルフリップ台） |
| camera_main | Camera | メインカメラ |
| light_main | Light | メインライト |

### オーバーレイUI

| 要素ID | 種類 | 説明 |
|--------|------|------|
| txt_title | Text | ゲームタイトル |
| txt_player_count | Text | 「接続中: X人」 |
| panel_recent | Panel | 最近のアクティビティ |
| txt_qr_code | Image | 接続用QRコード |
| txt_url | Text | 接続URL |

### アクティビティパネル

| 要素ID | 種類 | 説明 |
|--------|------|------|
| scroll_activity | ScrollView | アクティビティログ |
| item_activity | Prefab | アクティビティアイテム |

## 3Dシーン構成

```
Scene
├── Floor (床)
├── Table (テーブル)
│   └── SpawnPoint (ボトル生成位置)
├── Bottles[] (投げられたボトル群)
│   └── Bottle
│       ├── Model (3Dモデル)
│       └── CommentCanvas (コメント表示)
├── MainCamera
└── Lights
```

## 動作仕様

### ボトル表示
- プレイヤーがボトルを投げると3Dボトルが生成
- Rigidbodyでボトルフリップ物理演算
- 成功/失敗判定後、コメント付きで静止

### コメント表示
- ボトルの上にビルボード形式でコメント表示
- プレイヤー名 + コメント内容
- 一定時間後フェードアウト（設定可能）

### カメラワーク
- 通常時：固定カメラ
- ボトル投げ時：ボトル追従カメラ（オプション）

### 接続情報表示
- QRコードで接続URL表示
- 現在の接続プレイヤー数表示

## 演出

### ボトル投げ演出
- 投げ開始時：効果音
- 成功時：パーティクル、成功音
- 失敗時：残念音

### コメント演出
- フェードイン表示
- 吹き出しアニメーション
